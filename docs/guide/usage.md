# 如何使用本站

本站是基于 MkDocs Material 主题构建的 R 项目开发指南文档站点，旨在分享 R 语言项目开发的最佳实践和经验。

## 🔍 浏览指南

### 导航方式

#### 1. 侧边栏导航
- 点击左侧导航栏可以快速浏览所有指南分类
- 导航栏支持折叠/展开，方便查看文档结构
- 当前页面会在导航栏中高亮显示

#### 2. 顶部搜索
- 点击顶部搜索框或按 `F` 键激活搜索
- 支持全文搜索，输入关键词即可找到相关内容
- 搜索结果会显示匹配的上下文

#### 3. 目录导航
- 每个指南页面右侧显示该页面的目录结构
- 点击目录项可快速跳转到对应章节
- 目录会随页面滚动自动高亮当前位置

### 阅读功能

#### 主题切换
- 点击顶部的 🌓 图标可以切换深色/浅色模式
- 系统会记住你的偏好设置

#### 代码复制
- 代码块右上角有复制按钮，点击即可复制代码
- 支持语法高亮，便于阅读

#### 编辑建议
- 每个页面顶部有 "编辑此页" 链接
- 点击可跳转到 GitHub 对应的源文件
- 欢迎通过 GitHub 提交改进建议

## 🔖 实用技巧

### 快速定位

1. **使用搜索**: 不确定内容在哪？直接搜索关键词
2. **浏览目录**: 查看页面右侧目录，了解文档结构
3. **关注链接**: 文档内部链接可快速跳转相关内容

### 保存常用页面

将常用指南页面添加到浏览器书签，方便快速访问：

- R 项目结构指南
- R 编码规范
- 常用工具文档

### 分享给同事

每个页面都有独立的 URL，可以：

- 📧 直接通过邮件分享链接
- 💬 在团队聊天工具中分享
- 📋 添加到团队知识库

---

## ⚙️ 个性化设置

### 调整字体大小

使用浏览器的缩放功能（`Ctrl +/-` 或 `Cmd +/-`）调整页面字体大小。

### 打印页面

如需打印某个指南：

1. 使用浏览器的打印功能（`Ctrl+P` 或 `Cmd+P`）
2. 选择"另存为 PDF"以保存电子版
